# Logo Directory

Place the `lejgopro-email.png` logo file in this directory.

## Expected file:
- `lejgopro-email.png` - Logo for PDF invoices

## Usage:
The logo will be automatically included in PDF invoices generated by the system.

## Specifications:
- **Format:** PNG (recommended for transparency)
- **Size:** Recommended 200x100px or similar aspect ratio
- **Position:** Top-right corner of PDF invoices
- **Display size:** 40x20 points in PDF
